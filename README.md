# Application Hosting Operations

## Overview

This repository contains operational procedures, deployment guides, monitoring standards, troubleshooting methods, and recovery procedures for enterprise application hosting environments.

The repository focuses on Java-based application environments deployed on Apache Tomcat and Eclipse Jetty.

## Scope

- Apache Tomcat Administration
- Eclipse Jetty Administration
- Java Application Deployment
- Service Monitoring
- Performance Management
- Troubleshooting
- Backup and Recovery
- Operational Runbooks

## Technologies

- Java
- Apache Tomcat
- Eclipse Jetty
- Linux
- Windows Server
- PostgreSQL
- Monitoring Tools
